# Standardization
*Standardization:

*Introduction

Standardization is a data preprocessing technique used to transform numerical features into a common scale without distorting differences in the range of values.

It rescales the data so that:

Mean becomes 0

Standard deviation becomes 1

Standardization is commonly used in Machine Learning algorithms that are sensitive to feature scaling such as:

Linear Regression

Logistic Regression

KNN

SVM

PCA


*Objectives of Standardization:

1.Bring all features to a common scale

2.Improve Machine Learning model performance

3.Prevent features with larger values from dominating

4.Speed up gradient-based algorithms

5.Improve convergence during training


*Steps Involved in Standardization:

1. Import Required Libraries

2. Load the Dataset

3. Select Numerical Features

4. Apply Standardization

5. View Standardized Data


*Advantages of Standardization:

Improves model accuracy

Faster model training

Better convergence

Removes scaling issues

Useful for distance-based algorithms


*Disadvantages of Standardization:

Sensitive to outliers

Does not bound values to a fixed range

Interpretation becomes difficult


*Conclusion:

Standardization is an important preprocessing step in Machine Learning that transforms numerical data into a standard scale. It improves model performance, training speed, and helps algorithms perform efficiently on datasets with features having different ranges.
